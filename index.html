LICHeE: Lineage Inference for Cancer Heterogeneity and Evolution

Program: smuth (Somatic Mutation Hierarchies)
Usage:
 -build    Construct the sample cell lineage trees
 -i    Input file path (validation or vcf file)
 -o    Output file path (default: input file path)
 -n,--normal    Normal sample id
 -minAAFPresent   Minimum AAF to robustly call an SNV in a sample (default: 0.04)
 -maxAAFAbsent   Maximum AAF to robustly consider an SNV as absent from a sample (default: 0.015)
 -maxAAF    Maximum allowed AAF in a sample (default: 0.6)
 -minGroupSize   Minimum SNV group size (default: 1)
 -minRobustGroup   Minimum number of robust SNVs in group to be robust (default: 2)
 -minRobustGroupKeep   Minimum number of robust SNVs per group to keep the group in the network (default: 0 - keeps all the groups initially)
 -minTargetDistRatio   Minimum non-robust SNV to target group's SNV ratio per sample (default: 0.5)
 -e    AAF error margin (default: 0.1)
 -maxClusterDist   Maximum mean AAF difference on average per sample up to which two clusters can be collapsed (default: 0.1)
 -minClusterSize   Minimum size a cluster must have to be considered a node in the network (default: 2)
 -net,--showNetwork   Display the constraint network
 -closestParentOnly   Only add edges to the ancestor in the closest possible level
 -top    Number of top ranking lineage trees to display
 -tree,--showTree    Display the top ranking lineage tree
 -v,--verbose    Verbose mode
 -s,--save   Save the output to file
 -h,--help   Print usage
 -cnv   File path to CNV regions used to mark SNVs
 -tcga   File path to TCGA SNV annotations
 -cosmic   File path to COSMIC SNV annotations
 -ann   File path to ANNOVAR variant function annotations

HOW TO RUN
----------
./smuth -build -i - n [options]

EXAMPLES
--------
#Show the top ranking tree
./smuth -build -i ../data/ccRCC/RK26.txt -tree -n 0
#Eliminate clusters that have fewer than 3 SNVs
./smuth -build -i ../data/ccRCC/EV007.txt -n 0 -minClusterSize 3 -tree
#Change hard SNV caller thresholds
./smuth -build -i ../data/ccRCC/RMH004.txt -n 0 -minClusterSize 5 -maxAAFAbsent 0.005 -minAAFPresent 0.01 -tree
