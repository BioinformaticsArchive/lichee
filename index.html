LICHeE: Lineage Inference for Cancer Heterogeneity and Evolution

PROGRAM PARAMETERS

-build Construct the cell lineage trees
-i <arg> Input file path (validation or vcf file, see /data directory for examples)
-o <arg> Output file path (default: input file path)
-n,--normal Normal sample id
-minAAFPresent <arg> Minimum AAF to robustly call an SNV in a sample (default: 0.05)
-maxAAFAbsent <arg> Maximum AAF to robustly consider an SNV as absent from a sample (default: 0.02)
-maxAAF <arg> Maximum allowed AAF in a sample (default: 0.6)
-e <arg> AAF error margin (default: 0.1)
-minGroupSize <arg> Minimum SNV group size (default: 1)
-minRobustGroup <arg> Minimum number of robust SNVs in group to be robust (default: 2)
-minRobustGroupKeep <arg> Minimum number of robust SNVs per group to keep the group in the network (default: 0 - keeps all the groups)
-minTargetDistRatio <arg> Minimum non-robust SNV to target group's SNV ratio per sample (default: 0.5)
-maxClusterDist <arg> Maximum mean AAF difference on average per sample up to which two clusters can be collapsed (default: 0.1)
-minClusterSize <arg> Minimum size a cluster must have to be considered a node in the network (default: 2)
-closestParentOnly Only add edges to the ancestor in the closest possible level (default: false)
-net,--showNetwork Display the constraint network
-tree,--showTree Display the top ranking lineage tree
-top Number of top ranking lineage trees to display
-v,--verbose Verbose mode
-s,--save Save the output to file
-h,--help Print usage

HOW TO RUN

./lichee -build -i - n [options]

EXAMPLES

#Show the top ranking tree
./lichee -build -i ../data/ccRCC/RK26.txt -tree -n 0

#Eliminate clusters that have fewer than 3 SNVs
./lichee -build -i ../data/ccRCC/EV007.txt -n 0 -minClusterSize 3 -tree

#Change hard SNV caller thresholds
./lichee -build -i ../data/ccRCC/RMH004.txt -n 0 -minClusterSize 5 -maxAAFAbsent 0.005 -minAAFPresent 0.01 -tree
